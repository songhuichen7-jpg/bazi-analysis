# 八字工具内测 · 试跑指引

感谢你帮忙试跑。大约 15-20 分钟，跑完你把感受告诉我就行，不用写报告。

---

## 开始前

- 链接：**[填你的部署地址]**
- 准备：你真实的 **出生年月日、出生时辰（几点几分）、出生城市**
- 环境：**请在手机上跑一遍，再在电脑上跑一遍**（两端体验可能不一样，都想听）

---

## 你要做的事

**全程保持"小白心态"——看到不懂的词、点不动的按钮、不知道在哪的功能，都是我要的反馈。不要自己琢磨"是不是我没找到"，找不到本身就是问题。**

### 任务 1 · 填表排盘

1. 打开链接
2. 输入你的出生信息，点提交
3. **观察**：加载过程顺不顺？卡了多久？有没有转圈转到你想关页面？

### 任务 2 · 看主分析

1. 页面出来后，从上到下读一遍
2. **观察**：
   - 有哪几个字 / 词你看不懂？（比如"伤官配印""从格"这种，看不懂就是看不懂，直说）
   - 哪段读完觉得"这讲的就是我"？哪段觉得"在说什么"？
   - 有没有想看但没找到的东西？（比如"我想知道今年怎么样"找不到在哪）

### 任务 3 · 看古籍判词

1. 滚到"古籍判词"那一栏
2. 随便点一条展开看看原文
3. **观察**：
   - 原文看得下去吗？白话解释有帮助吗？
   - 判词和你本人对得上吗？有没有"这也太准了"或者"完全不像我"的？

### 任务 4 · 聊天提问（重点）

在对话框里问至少 **3 个问题**，越刁钻越好。比如：

- "我今年换工作合适吗"
- "我什么时候结婚"
- "我适合做什么行业"
- 或者你真正好奇的任何问题

**观察**：
- 回答有没有答非所问？
- 有没有哪句话里带方括号 `[[...]]` 的"小标签"？点一下看看会发生什么。
- 第二个问题问下去，它还记得你是谁吗？

### 任务 5 · 起卦（可选，看到再跑）

页面里如果有"起卦"或者"占卜"相关的按钮，点一下试试，看出来的东西能不能看懂。

### 任务 6 · 换设备

**电脑跑完请换手机再跑一次**（或反过来）。

**观察**：
- 手机上字会不会太小 / 按钮点不准 / 滚动错位？
- 有没有哪个功能手机上找不到但电脑上能看到？

## 跑完给我反馈

最简单的格式（微信随便发都行）：

```
【卡住的地方】
·

【看不懂的词】
·

【想看但没找到】
·

【最喜欢 / 觉得准的地方】
·

【手机 vs 电脑差异】
·
```

**能截图的地方尽量截图**，特别是出错 / 加载慢 / 排版奇怪的瞬间。

---

## 不用做的事

- 不用帮我想"怎么改"——反馈原始感受就够了
- 不用客气说"挺好的"——直接说哪里烦
- 不用跑完所有功能——跑到你不想跑了就停，告诉我停在哪
