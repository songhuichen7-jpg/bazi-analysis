# Synthesizer 反偏差专项

> 这份文件由 SKILL.md 引用。包含从实战中发现的 8 个 synthesizer 系统性 bug、防护铁律、和内部 checklist。
>
> **重要**:本文件的所有 checklist 和铁律是 synthesizer **内部使用**的,**不要写在给用户的输出里**(参见 SKILL.md §0.5)。

---

## 起因(为什么有这份文件)

实战回测中反复出现 8 类**系统性**(不是个别)bug。所有 bug 都不是命盘错,而是 synthesizer 在汇总阶段**偷懒**——力量擂台跑了,但白话翻译时只看了主流力量,忽略了细节。

这份文件的目的:**让以后任何 synthesizer 在白话翻译之前,跑一遍这个 checklist,避免犯同样的错**。

---

## 1. 两种 bug 类型(原始分类)

### 类型 A: 正/偏对子混淆

笼统说"X 旺/X 弱",没有区分正/偏。BaZi 中每对正/偏十神**作用方向完全不同**,不能合并:

| 对子 | 正 vs 偏的差异 | 不细分会犯的错 |
|---|---|---|
| **正官 vs 七杀** | 正官=主动认同的服从;七杀=压力下的服从 | 看正官弱 → 错断"不守规矩"。实际七杀压制 + 食伤近零 + 比劫弱 → 仍极度被动守规矩 |
| **正财 vs 偏财** | 正财=稳定/必需/日常;偏财=投机/波动/机会型 | 看偏财旺 → 错断"对所有想要的东西都失重"。实际是金融/投机失重,日常物质可控 |
| 正印 vs 偏印 | 正印=正统传承;偏印=偏门吸收 | 看印总量 → 错估"学习能力" |
| 食神 vs 伤官 | 食神=温和创造;伤官=锋利批判 | 看食伤总量 → 错估"表达类型" |
| 比肩 vs 劫财 | 比肩=平等同党;劫财=争夺型同党 | 看比劫总量 → 错估"合作模式" |

**铁律**:**力量擂台后,synthesizer 必须显式比对每对正/偏对子,不能笼统说"财旺/官旺/杀旺/印旺/食伤旺/比劫旺"**。每条行为预测必须指明它依赖的是正/偏中的哪一个。

### 类型 B: 功能近邻十神混淆

把"功能上相似但作用方向不同"的十神当成同一回事。**最常见的混淆对**:

| 易混淆对 | 真实差异 | 不分会犯的错 |
|---|---|---|
| **食伤 vs 比劫** | 食伤=对外的表达通道;比劫=对内的自我意识 | 看食伤近零 → 错断"无叛逆/无自主"。实际比劫支撑下仍有"安静的自主决定" |
| 印 vs 比劫 | 印=被动接受滋养;比劫=主动同党 | 看印弱 → 错断"无支持系统",忽略比劫的自我支撑 |
| 财 vs 食伤 | 财=对外的占有;食伤=对外的输出 | 都标记为"对外"力量,但一个是抓取,一个是给予 |
| 官杀 vs 印 | 官杀=约束;印=吸纳约束并转化 | 看官杀旺以为"压力大",忽略印能否化杀 |

**铁律**:**任何"X 近零所以无 Y"的推理,必须先确认 Y 的真正主要来源是不是 X。**

### 食伤 vs 比劫 专项辨析(实战出错最多)

实战中**食伤 vs 比劫的混淆**出现频率最高,且容易在同一盘上反向出错两次。两者都属于"日主同党/支持系",但功能完全不同:

| 维度 | 食伤(食神/伤官) | 比劫(比肩/劫财) |
|---|---|---|
| 方向 | **对外**——把日主的能量"吐"出去 | **对内**——构建日主的"自我"边界 |
| 表现 | 表达、创作、说话、做作品、批判、生产 | 自我意识、独立、按自己意愿、不被替代 |
| 缺失时 | 无法对外表达自我 → 大众场合大脑空白、无创作出口 | 无自我边界 → 容易被他人同化、无独立判断 |
| 旺盛时 | 能输出、有创造力、爱表达 | 有主见、独立、可能不合群 |
| 关键区分 | "能不能说出来/做出来" | "知不知道自己要什么" |

**典型组合**:

| 食伤 | 比劫 | 行为预测 |
|---|---|---|
| 旺 | 旺 | 主动表达 + 强自我 → 典型外向独立型 |
| 旺 | 弱 | 能表达但易随大流 → 表演型/迎合型 |
| **弱** | **弱** | **既不能表达又无自我** → 真正的"被驱使型" |
| **弱** | **有(中弱)** | **不能对外表达,但内心有清晰自我决定 → "安静的自主主义者"** |
| 弱 | 强 | 沉默但极有主见 → 典型沉默型独立人 |

**铁律**:**食伤近零时,不能直接断"无叛逆/无自我"。必须先单独评估比劫,因为"想走自己路"的主要来源是比劫不是食伤。**

## 2. 类型 C: 主题锚定偏差(响应偏差)

**定义**:Synthesizer 在响应用户特定主题(如"财运"、"事业"、"婚姻")提问时,**潜意识把所有命盘元素都向用户问的主题倾斜**,即使有些元素的真正主题不在那里。

**典型场景**:
- 用户问"财运"
- Synthesizer 把**所有大运段**都向"财"框架解读
- 把晚年印运框成"享受财 / 传承财"
- 实际上印运的真正主题是"自我被滋养、慢下来",不是财
- 命主回测时质疑"为什么年纪这么大还跟财关联那么大"——bug 暴露

**bug 1-4 vs bug 5 的差异**:
- bug 1-4 是**力量擂台漏判**(类型 A 正/偏混淆,类型 B 功能近邻混淆)
- bug 5 是**响应偏差**——不是没看清力量,是被用户提问"拉走"
- 这是 LLM 在响应时的典型陷阱:**响应式锚定**

**铁律**:**Synthesizer 在响应用户特定主题提问时,必须先做"主题 vs 底色"区分**:
- **底色**:命盘里一直存在的结构特征(如"地支三土→财是终身底色")
- **主题**:某段时期的核心矛盾(如"67-87 印运主题是滋养",不是"财")
- **不是命盘里所有含 X 元素的部分都是 X 主题**
- 每个大运/流年应**独立判断**它的真实主题,而不是被用户提问 anchoring

**Synthesizer 必问**:"用户问的是 X,但我现在分析的这段大运/流年的**真实主题**是不是 X?如果不是,我必须明确说'这段是底色不是主题'。"

## 3. 十神的多重含义与维度强度差异

### 十神的多重含义对照表

**任何把十神简化为单一含义的解读都会失真**。

| 十神 | 字面含义 | 实际指向(部分清单) |
|---|---|---|
| **正财** | 钱(稳定) | 工资、稳定收入、妻(男命)、给予的资源、固定关系 |
| **偏财** | 钱(灵活) | 副业、投机、机会、信息、人脉、感官享受、流动资源、非妻女性(男命)、父(部分流派) |
| **正官** | 官 | 主动认同的规则、正职、丈夫(女命)、社会认可 |
| **七杀** | 杀 | 被动承受的压力、外部权威、竞争、挑战、情人(女命) |
| **正印** | 印 | 母(部分流派)、学问、传统、被动接受的滋养、稳定支持系统 |
| **偏印** | 偏印 | 偏门学问、宗教、玄学、独特技能、不寻常的滋养源、父(部分流派) |
| **食神** | 食 | 温和表达、享受、孩童式好奇、生发、子女 |
| **伤官** | 伤 | 锋利批判、艺术、突破规则、聪明、不服管 |
| **比肩** | 比 | 平等同党、自我意识、合作伙伴 |
| **劫财** | 劫 | 争夺型同党、竞争对手、争财对象 |

**铁律**:**任何"X 十神 = Y 单一含义"的判断都是错的**。十神是**抽象力量类**,具象化在不同维度有不同形态。Synthesizer 必须明示"我现在用的是 X 的哪个具体含义"。

### 维度强度差异原理

**核心洞察**:某十神的力量是固定的(如偏财 8.5/10),但**它在不同维度的兑现强度由"其他十神/结构因素"调节**。

```
某十神原始力量
    ↓
被调节因素影响:
    ├── 日主强弱(决定能否承受)
    ├── 同类其他十神(增强或互补)
    ├── 邻位十神(中介或阻断)
    ├── 地支藏干(库特性、根的稳定度)
    ├── 大运补给(临时改变)
    ↓
不同维度的实际兑现强度
```

**示例**(假设某盘偏财力量 8.5,身弱+食伤近零+印旺):

| 维度 | 兑现强度 | 调节因素 |
|---|---|---|
| 金融/投机 | **9/10** | 几乎无调节,身弱 + 食伤近零 + 印来不及反刍 |
| 机会"看见"敏感度 | 8/10 | 偏财本能 |
| 关系内心吸引 | 7/10 | 拉力强,但被食伤近零外在压制 |
| 信息流 | 6/10 | 印的反刍部分介入 |
| 物质享受 | 6/10 | 财库收纳 + 印介入 |
| 跨界多元 | 6/10 | 印让多元变深度 |
| **机会"抓住"能力** | **4/10** | 身弱 + 食伤近零 |

**铁律**:**当用户问"X 十神(力量 N/10)对我的影响"时,Synthesizer 不能只回答"N/10 → X 影响"**。必须:
1. 列出 X 十神的多个含义维度
2. 对每个维度评估调节因素
3. 给出"维度 × 强度"的矩阵
4. 明示哪些维度是"高失重/高显化",哪些是"被调节弱化"
5. **同一个十神在不同维度的兑现强度可以差 4-5 分**

**最常见的"分裂结构"**:
- "看见 vs 抓住"分裂:偏财让你看见多,但身弱让你抓不住
- "内心 vs 外在"分裂:偏财让你内心被吸引,但食伤近零让你外在表达不出来
- "原局 vs 大运"分裂:原局某通道堵,大运临时打开

**Synthesizer 必须**显式呈现这些分裂,**不能**假装"力量 N → 影响 N"的线性关系。

## 4. 类型 F: 合/克/生关系遗漏(v1.3 新增)

**核心问题**:力量擂台只看每个十神的"量"(力量数值),不看"质"(与日主的合/克/生关系)。**两者都重要**——量决定影响大小,合/克/生决定影响**形态**。

**典型场景**(以"日主甲木 + 多土财"结构为例):
- 地支多土 → 财是终身底色
- 命主回测时反问:"前面分析里财都偏物质,缺少'情'的维度?"
- 检查发现:**日主甲与偏财戊是"克"关系(无合),与正财己只有藏干暗合(弱)**
- → **整盘日主-财关系基本是纯克,无显合**
- → 财在结构上偏"占有/功能性",缺少"情感融合"的维度

**铁律**:**力量擂台后,必须显式跑"日主与每个十神的合/克/生关系"分析**。
- **量**(力量数值):决定影响的大小
- **质**(合/克/生):决定影响的形态
  - **合**(融合,有"情"维度)
  - **克**(占有/主宰,纯功能)
  - **生**(滋养,被动接受)
- **同样力量的财**,在"合"vs"克"下的体验完全不同——前者有情感融合,后者只是占有
- 同理适用于官杀(合 vs 克)、印(生 vs 不生)、食伤、比劫

**Synthesizer 必问**:"日主与这个主导十神是什么关系?克?合?生?如果是纯克,这个十神在命主生活中的体验是**功能性**的,不是**情感性**的——这个区分必须显式说出来,不能默认假设。"

## 4A. 类型 G：偏财替代表达通道遗漏（v1.6 新增）

**核心问题**：类型 B 已指出"食伤近零 → 不能断无叛逆，比劫是替代来源"。但还有**第二个替代通道被遗漏**：**偏财旺 → 对外感知强 → 可替代部分表达功能**。

**偏财与食伤的功能差异**：

| 维度 | 食伤 | 偏财 |
|---|---|---|
| 对外表达类型 | 主动输出（说/写/做） | 感知/接收/社交互动 |
| 表达驱动 | 内发型（我有话要说） | 外发型（外部信息激活我） |
| 缺失时 | 没话说，大脑空白 | 对外部环境不敏感 |

**食伤近零 + 偏财旺 时的正确推论**：
- ❌ 错误：食伤近零 → 表达出口完全封闭
- ✅ 正确：食伤近零 → 主动输出受限；但偏财旺 → 对外感知丰富、社交信息灵敏 → 表达通道换了形式（感知驱动型，而非创作输出型）

**铁律**：**食伤近零时，必须同时检查偏财力量**。偏财旺 → 表达出口换了赛道，不是"无"。

## 4B. 类型 H：五行属性直接映射行为特质（v1.6 新增）

**核心问题**：把日主的五行属性（金/木/水/火/土）直接映射到性格/行为，而不做十神力量对比。

**常见错误**：

| 错误推论 | 错误原因 |
|---|---|
| "庚金日主 → 性格刚硬木讷" | 五行属性 ≠ 行为；行为取决于十神力量 |
| "壬水日主 → 圆滑流动" | 同上 |
| "甲木日主 → 向上生发" | 同上 |

**正确做法**：五行属性描述日主的"本质倾向"，但**实际行为由十神力量决定**：
- 庚金日主 + 偏财旺 + 食伤强 → 社交活跃，不一定"刚硬"
- 庚金日主 + 七杀旺 + 食伤近零 → 被压制，行事谨慎

**铁律**：**"X五行属性 → Y行为"的推理链，只有1步，但仍然是错的。必须从十神力量分布推导，五行属性只作类象参考，不作行为预测依据。**

## 5. Synthesizer 内部 checklist(每次综合前必跑,但不外显)

**重要**:这个 checklist 是 synthesizer **自己跑的**——**结果直接体现在回答的质量里**,**不要写在给用户的输出里**(参见 SKILL.md §0.5)。

按以下顺序 1-12,任何一条违反 → 该结论需要重做:

1. ☐ 我有没有在用"X 旺/X 弱"的笼统表述,而没有指明是正还是偏?
2. ☐ 我推某个行为预测时,有没有把功能近邻的十神(食伤 vs 比劫、印 vs 比劫、财 vs 食伤)混在一起?
3. ☐ 我说"X 近零所以无 Y"时,Y 的真正来源**是不是** X?(还是其他十神?)
4. ☐ 我有没有因为某十神最高就把整个相关行为归到它,而忽略了其他来源?
5. ☐ 我的白话翻译有没有暗中升级置信度?(把"倾向于"翻译成"会"或"是")
6. ☐ 如果用户回测说"不符合",我的第一反应是不是检查 synthesizer bug,而不是怀疑命盘?
7. ☐ 我是不是在死板套用某条 §4 风格的反误读规则,而没看具体力量分布?
8. ☐ 我有没有跑"日主与各十神的合/克/生"关系分析?(类型 F)
9. ☐ 我是不是被用户提问的主题"拉走"了,把所有元素都向那个主题倾斜?(类型 C)
10. ☐ 食伤近零时,我有没有同时检查偏财力量?偏财旺 → 有替代表达通道,不能断"无表达出口"。(类型 G)
11. ☐ 我是不是在用五行属性(金/木/水/火/土)直接推行为特质,而没有做十神力量对比?(类型 H)
12. ☐ 我有没有在定格局之前就分析行为?(§2.1 格局是地基,十神行为分析要建在格局之上)

## 6. Bug 模式速查(脱敏后的抽象案例)

| Bug | 类型 | 错误模式 | 修正方向 |
|---|---|---|---|
| **1. 守规矩** | A | "正官弱 → 守规矩力量弱" | 七杀强 + 食伤近零 + 比劫弱 → 仍然极强被动守规矩 |
| **2. 内向** | A | "身弱不决定内外向 → 想冲就冲" | 食伤近零 + 比劫弱 + 七杀压制 + 偏财旺 → 慢热型内向 |
| **3. 失重** | A | "偏财旺 → 对想要的东西失重" | 必须分正/偏:投机失重、物质混合、关系冷静各自不同 |
| **4. 叛逆** | B | "食伤近零 → 结构性无叛逆" | 比劫有根 → 仍有"安静的自主决定" |
| **5. 主题锚定** | C | 把晚年印运框成"享受财" | 印运主题是滋养,财是底色不是主题 |
| **6. 单一含义** | 派生 | 把"偏财"等同于"金钱" | 偏财指向多个维度,金钱只是其一 |
| **7. 强度均一** | 派生 | 假设某十神在所有维度强度均一 | 不同维度兑现强度被调节,差异可达 4-5 分 |
| **8. 合/克遗漏** | F | 财都偏物质,没区分合 vs 克 | 日主与财纯克无合 → 财缺"情"的维度 |

**这 8 个 bug 的共同模式**:
- 都发生在多 agent 分析后的 synthesizer 汇总阶段
- 都是 synthesizer 在白话翻译时**只看一个十神的力量数值,没看对子的全部成员或合/克/生关系**
- 都能被命主回测**直接证伪**
- 修正后通常**全部命中**(因为正确推导的依据**本来就在原始力量擂台里**,不是新发现)

→ **元层教训**:命盘没错,synthesizer 偷懒时错。**不要在每次输出里写"反方对抗自查",但要在内部跑这个 checklist**。

## 7. 何时回看本文件

Synthesizer 应该在以下时机内部跑一遍这个 checklist:

1. **完成力量擂台后,白话翻译之前**——避免类型 A/B
2. **响应用户特定主题问题前**——避免类型 C
3. **回答涉及某个十神的影响时**——避免十神单一含义化
4. **回答涉及力量"大"的十神时**——避免假设维度强度均一
5. **完成性格/事业/财运/关系分析后**——确认是否跑了合/克/生
6. **被用户回测推翻某条结论时**——首先检查是不是这 8 类 bug 之一
