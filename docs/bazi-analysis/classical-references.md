## 2A. 经实战验证的经典结论

以下是从经典中提取、经过回测验证的关键判断依据。可直接引用(来源已注明),无需再次 Read 原文。

### 三刑被两部顶级经典降级

| 经典 | 原文 | 态度 |
|---|---|---|
| **子平真诠** | "至于三刑取庑,**姑且阙疑**,虽不知其所以然,于命理亦无害也" | 搁置存疑 |
| **滴天髓** | "支神祇以**冲为重**,刑与穿兮**动不动**" | 明确降级:冲 > 刑 |

**实战结论**:分析地支关系时,冲为主要动力,刑为叠加因素。三刑不应作为独立判断依据。

### 冲的方向性

**滴天髓**:"旺者冲衰衰者拔"

冲不是双方等量对抗,而是旺的一方拔动衰的一方。判断谁受损需看:月令谁得令、四柱中谁的根多,得令+有根者胜。

### 外貌判断的正确典籍权威

**三命通会·论六亲**:"妻妾星坐生旺 → 主**美貌多才**"

滴天髓任铁樵否定"桃花判**休咎(吉凶)**" → 不是否定"桃花判外貌"。两个领域权威不同,不可混用。

### 子女星的学派分歧

| 体系 | 子女星 | 来源 |
|---|---|---|
| 早期命理旧说 | 正官=女儿、七杀=儿子 | 渊海子平体系；**子平真诠第24章明确否定此说**（"时说以讹传讹"） |
| 子平真诠体系 | 以宫分配六亲，子息看时柱 | 子平真诠第23-24章 |
| 滴天髓体系 | "今更定以**食伤为子女**" | 任铁樵 |
| 任铁樵变通法 | "身旺财为子,身衰印作儿" | 任铁樵 |

**实战建议**:子女分析必须标注使用的体系；"正官=女儿/七杀=儿子"是古旧说法，子平真诠已否定，不应作为主要依据。置信度因学派分歧而降低。

### 妻星 = 干头之财

**子平真诠第二十四章**:"妻星者,**干头之财**也" → 天干透出的财=妻星。

当天干透出的是偏财而非正财时产生张力:正财为妻(基本原则);"干头之财"解释了偏财型异性为什么"先出现"——不代表是最终伴侣。

### 穷通宝鉴的"输情"

穷通宝鉴对特定日干的描述常含深层信息。例如"**庚必输情于乙妹**"——此语出自穷通宝鉴**乙木章七月节**（分析乙木在申月的情形），描述乙庚六合中庚的主观倾向。"输情"比"合"更强烈，带有主观意愿的"倾心"含义。

**实战建议**:查穷通宝鉴时不只看调候用神列表，要逐字读原文注释；"输情"在乙木章，不在庚金章。

### 滴天髓夫妻章要点（体系分歧警告）

任铁樵《滴天髓阐微·夫妻章》与子平真诠体系在夫妻六亲分析上存在根本性分歧：

| 要点 | 说明 |
|---|---|
| 否定"阳刃劫财主克妻"旧说 | 任铁樵明确批驳以阳刃劫财机械推断"克妻"，认为须看整体格局气势 |
| 六亲体系不兼容 | 滴天髓以格局气势为核心分析夫妻，不套子平真诠的"宫分配六亲"规则 |
| 实战建议 | 关系动态→滴天髓夫妻章；宫位配六亲→子平真诠第23/24章；两套体系并列时，明确告知用户存在体系分歧 |

---

### 贪合忘生/克/冲

| 经典来源 | 原文 | 原理 |
|---|---|---|
| 子平真诠·阳刃章 | "丁与壬合,则七煞有**贪合忘克**之意" | 合了就忘了原本的克制功能 |
| 滴天髓案例 | "若邀庚金生助,**贪合忘生**" | 合了就忘了原本的生助功能 |
| 滴天髓案例 | "甲以乙妹妻庚,凶为吉兆,**含合忘冲**" | 合可以化解冲的威胁 |

**应用**:当A与B合时,A可能"忘记"它本来对C的生/克/冲/刑功能。这可能截断威胁链。

### 桃花/神煞的降级

**滴天髓任铁樵**:"荒唐取用,桃花咸池...乃好事妄造,**非关命理休咎**"

否定的是"桃花判吉凶",不是"桃花判外貌"。神煞在外貌领域(三命通会体系)仍有独立权威。

## 2B. 经典查阅决策流程

分析任意命盘时，按以下 6 步顺序查阅，不要跳步：

```
第一步：排盘验证
  四柱、十神是否正确？不确定 → 查《渊海子平》干支体象章确认定义

第二步：格局判断（SKILL §2.1）
  月令本气透干 → 定格局名 → 查《子平真诠》第8-12章判断成败条件

第三步：调候分析
  日主 × 月令 → 查《穷通宝鉴》对应日干章节（逐字读原注！）

第四步：整体气势
  身旺/身弱？从格？合化条件？→ 查《滴天髓阐微》衰旺/形象章节

第五步：细节补充（按需）
  外貌 → 三命通会·卷07（论性情相貌）
  神煞 → 三命通会·卷03
  日时特殊组合 → 三命通会·卷08-09（日时断）
  纳音 → 三命通会·卷01（纳音章）

第六步：交叉验证
  关键结论在多典籍方向一致 → 高置信度
  有分歧 → 明确告知用户"古籍内部有争议"，降级置信度
```

**必须查阅的场景**（不查=高风险出错）：

| 场景 | 必查典籍 |
|---|---|
| 判断格局是否成立 | 子平真诠第8-12章 |
| 判断日干在该月的喜忌 | 穷通宝鉴·对应日干章节 |
| 判断天干合是否化 | §6.1 三条件 checklist + 子平真诠/滴天髓 |
| 判断冲的方向/力度 | 滴天髓"旺者冲衰衰者拔" |
| 判断配偶外貌 | 三命通会·卷07（论性情相貌） |
| 发现日时组合特殊 | 三命通会·卷08-09（日时断） |

**可以不查阅的场景**：D级推测（具体品牌/身高）、纯叙事描写、命主已确认的事实。

## 2C. 辅助信号的局限性

### 纳音

三命通会卷一对纳音（如松柏木、泉中水、白蜡金等）有专门论述。纳音不影响格局判断，但提供额外的性格/命运色彩参考。**作为辅助信号，不作为主要判断依据**。当主流分析不足以解释某点时，可查三命通会卷01纳音章作为补充。

### 日时断的局限性

三命通会卷08-09的日时断歌诀是对**所有**该日时组合的人的概括，不考虑年柱和月柱差异，用词概括性强（"妻财损"≠妻子会离开）。**必须结合完整四柱做具体分析，不能直接作为个人精准判断**。

### 互换贵禄等特殊格局

三命通会记录了子平真诠/滴天髓不一定提及的特殊格局（互换贵禄、飞天禄马等）。遇到日时组合特殊时，查三命通会对应章节，避免遗漏主流分析看不到的格局。

## 3. 引用格式(标准模板)

任何古籍引用必须遵循以下结构,**4 个步骤缺一不可**:

```markdown
**[书名] · [章节]** 说:

> [古籍原文]

**白话**:[一句话翻译]

**现代 reframe**:[去掉封建框架,转换成现代语境的解读]

**对照本盘**:[这条在命主身上是命中 / 部分命中 / 不符 / 古籍未涉及具体情况]
```

### 示例(展示引用格式,非真实命盘)

**穷通宝鉴 · 七月甲木** 说:

> 七月甲木,丁火为尊,庚金次之,庚金不可少。火隔水不能熔金,故丁火熔金,必赖甲木引助,方成洪炉。若有癸水阻隔,便灭丁火,壬水无碍,且能合丁,但须见戊土,方可制水存火。

**白话**:申月生的甲木最需要的是丁火(锋利的表达力),其次是庚金(外部压力源/挑战)。庚金本身不能少,但只有丁火才能"炼化"庚金。如果有癸水隔在中间,丁火就被灭掉了;戊土能挡住癸水,所以必须有戊土。

**现代 reframe**:此类命主最需要的是"通过锋利的、批判型的、突破性的表达"来处理外部压力。这个表达力如果出不来,只能被动承受压力,无法转化。"癸水(印)阻隔丁火"在现代语境是"过度反复思考反而阻碍行动";"戊土(偏财)能制癸水"在现代语境是"实际行动 / 物质实践能打破思考瘫痪"。

**对照命盘**(逐条核对四柱中是否出现以下用神):
- 庚金:是否透干?(古书"庚金不可少")
- 戊土:是否透干?(古书"须见戊土")
- 丁火:是否透干?藏支?(若不透 → "用神弱"的具体命理依据)
- 癸水:是否透干?(若透 → 古书警告"癸水阻隔便灭丁火",印反而干扰丁火)

→ 根据实际命盘逐项打勾/打叉,标注哪一条是该命主的关键缺口或亮点。

## 4. 现代 reframe 的方法

古籍写于明清时期(甚至更早),很多说法已经过时:

| 古籍说法(原意) | 现代 reframe |
|---|---|
| 妻子(男命的正财)| 长期稳定关系 / 配偶 |
| 妾(男命的偏财) | 流动型关系 / 非长期对象 / 也指"情感投入对象" |
| 富贵 | 物质成就 + 社会地位的综合 |
| 贫贱 | 资源不足 + 社会地位低,**不一定是钱的多少** |
| 田园广盛 | 资产 / 不动产 / 长期沉淀 |
| 科甲 | 进入主流认可体系(放今天可能是顶尖学校 / 大公司 / 公职) |
| 武职 | 需要执行力和对抗的工作(放今天可能是销售 / 创业 / 体育 / 军警) |
| 僧道清高 | 选择"非主流路径" / 内向型职业 / 学术 / 修行 |
| 克子刑妻 | 关系紧张 / 难以建立稳定家庭 — **不一定字面意义** |
| 早年身孤 | 童年/青年期独自承担多 / 缺乏支持系统 |
| 中年发福 | 中年阶段事业 / 财务有显著上升 |

**核心原则**:**保留古籍的"结构判断"(谁强谁弱、谁来谁去、利弊方向),去掉"具体形态描述"(妻、妾、田园、官职等)**。

## 5. 处理古籍冲突

不同古籍可能互相冲突。比如:
- 子平真诠说"七杀格喜食神制"
- 滴天髓说"身弱七杀宜印化优于食制"

这种冲突**不是 bug,是 feature**。处理方式:

1. **先看冲突在哪个维度**:是"用神选择"冲突?还是"格局判定"冲突?
2. **看哪个说法更针对此命盘**:子平真诠的"喜食制"是针对一般七杀格;但本盘食伤近零,所以子平真诠的建议无法实现 → 滴天髓的"宜印化"更适用
3. **明确告诉用户存在不同流派的观点**:不要装作只有一种答案
4. **回测验证**:如果命主提供了过去事件,看哪个流派的预测更准

**铁律**:**遇到古籍冲突时,必须告诉用户"古籍内部本身有争议",不要包装成"我有定论"**。

## 6. 古籍 vs LLM 推论的优先级

| 情况 | 优先级 |
|---|---|
| 古籍直接说了同一情况 | **古籍优先**(LLM 只用来翻译和 reframe) |
| 古籍讨论了相关原理但没说具体情况 | **古籍 + LLM 外推**(明确标注) |
| 古籍完全没涉及(比如现代职业策略、机制对策、心理学) | **LLM 推论**,但要明确标注"古籍没有直接说,这是基于结构的现代推论" |
| 古籍说法跟回测命中事实冲突 | **回测优先**(古籍可能基于的是不同时代的语境,需要 reframe) |

## 7. 如何引用(具体步骤)

当 synthesizer 在做分析,需要引用古籍时,**实际操作**是这样的:

### Step 1: 决定要引哪本书

根据"何时引用什么书"的表(本文档第 2 节)。

### Step 2: Read 对应文件

用 Glob 工具搜索章节文件，获得实际路径，再用 Read 工具读取。

**例**：分析"申月甲木的调候用神" → Glob(`**/classics/qiongtong-baojian/02_lun-jia-mu.md`) → 得到路径 → Read 该路径

不确定文件名时：先 Glob(`**/classics/[书名]/00_mu-lu.md`) → Read 目录文件 → 找到对应章节文件名。

### Step 3: 在文件中找到对应小节

每个章节文件内按月份/段落组织。比如 `02_lun-jia-mu.md` 包含三春甲木 / 三夏甲木 / 三秋甲木 / 三冬甲木 等小节。**找到与命主月令对应的那一段**。

### Step 4: 引用 + 翻译 + reframe + 对照

按本文档第 3 节的标准模板。

### Step 5: 不要捏造不存在的古籍内容

**绝对禁止**:
- ❌ 凭印象引用古籍但没真的读
- ❌ 把 LLM 训练里的命理常识包装成"古籍说"
- ❌ 编造书名 / 章节名 / 古文段落

**正确做法**:
- ✓ Read 文件 → 找到原文 → 复制原文 → 翻译 → 对照
- ✓ 如果找不到对应内容,说"古籍此处没有具体说明" + 标注是 LLM 推论

## 9. 何时不需要古籍

不是所有分析都需要引古籍。以下情况**可以**不用:

- **闲聊式问答**(比如"我大概是什么性格")
- **机制对策类**(如"机制 6 领域")—— 这是现代生活智慧,古籍没有
- **回测验证**(命主报告事件,对照命盘是否预测到)—— 这是体系验证,不是初次推导
- **现代职业规划**—— 古籍没有"做内容创作还是做工程"的概念
- **心理学层面的解读**(比如二阶痛苦、自我认知)—— 现代心理学概念

但**只要有可能引古籍**(尤其涉及命理原理、用神、格局、力量判断时),**就应该尝试引用**——这是 v1.4 的核心改动。

---

## 10. 古籍引用 ≠ 古籍崇拜

**最后的提醒**:引古籍是为了**给分析更扎实的根**,**不是**为了让分析显得权威。

- 古籍**会出错** —— 1000 年前的人对人性观察很深,但他们没有现代心理学
- 古籍的**封建框架**(妻妾、官职、富贵)需要 reframe
- 古籍的**矛盾**(不同流派之间)需要明确告知用户
- 古籍的**不完整性**(没有现代职业、心理、财务概念)需要 LLM 补充

**正确的态度**:**古籍是锚点,不是终点**。分析的最终目的是对命主有用,不是引经据典。
