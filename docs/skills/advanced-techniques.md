## §11 歌曲/作品匹配验证法

一种独特的回测验证方法:**如果命理推导出的"关系形状"是准确的,那么描述类似"形状"的歌词/作品应该自然匹配**。

### 方法

1. 从命盘推导出关系的"抽象形状"(如:寻找 → 锁定 → 融合 → 稳定)
2. 寻找歌词意象与此形状同构的作品
3. 逐句对照 → 标注匹配和不匹配之处
4. 匹配度 80%+ → 侧面验证推导的有效性

### 为什么歌曲能匹配而电影不能

- 歌曲和命理在同一抽象层面工作:描述"状态/形状"而非"事件/画面"
- 电影需要"可视的"破损和修复 → 如果命主的"不完整"是不可见的内在状态变化 → 电影拍不出来
- 歌曲不需要"看见" → 只需要"感受到" → 因此能达到更高的匹配度

## §12 六合化气与正缘意义分析

### 适用场景

当命盘中存在"日主与某十神形成天干六合"的结构时,需要分析该六合对命主生活的实际影响。

### 六合化气的分析步骤

1. **确认合的双方**:哪两个天干?分别代表什么十神?
2. **确认化气方向**:化成什么五行?该五行对日主是利是弊?
3. **评估合化条件**:月令是否支持化气?地支是否有化气之根?
4. **确认合化时机**:在哪个大运/流年 → 合化条件被满足?
5. **推导合化效果**:对日主的力量、性格表现、关系动态有何改变?

### 需要注意的分析陷阱

| 陷阱 | 说明 |
|---|---|
| 见合就化 | 有合不一定化 → 需要月令和地支支持 → 不满足条件的合只是"合而不化"(吸引但不转化) |
| 忽略合化的时间性 | 原局有合 → 但化气可能要等特定大运/流年才成 |
| 混淆"合"与"好" | 合化不一定对日主有利 → 化出的五行如果是忌神 → 反而有害 |
| 过度叙事化 | 六合的效果应以命理结构为主 → 不应过度投射情感故事 |

### 当六合涉及正财(男命=正缘)时的通用框架

**合化对日主有利时**(化出的五行补身或生身):

| 维度 | 分析方向 |
|---|---|
| 日主变化 | 日主力量从"弱"向"强"方向偏移 |
| 财务影响 | 身强 → 能"接住"财 |
| 事业影响 | 身强 → 能承受官杀压力 |
| 心理状态 | 从"搜索"模式 → "安定"模式 |

**合化对日主不利时**(化出的五行是忌神):

| 维度 | 分析方向 |
|---|---|
| 日主变化 | 日主力量被进一步耗泄 → 可能更弱 |
| 关系影响 | 合而化忌 → 婚姻关系可能消耗命主 |
| 财务影响 | 身更弱 → 更接不住财 |

### 使用原则

- 此框架需要结合具体命盘使用 → 不能脱离四柱和大运空谈
- 化合的利弊取决于具体五行关系 → **不存在"六合一定好"的通则**
- 分析时务必先做十神力量对比(SKILL.md §3)→ 再判断合化效果

## 大运起运公式

### 顺行 / 逆行

- **男阳年 / 女阴年** → **顺排**
- **男阴年 / 女阳年** → **逆排**

阳年阴年:年干为 甲丙戊庚壬 = 阳;乙丁己辛癸 = 阴

### 起运岁数(精确)

从出生日往前(逆排)或往后(顺排)数到最近的节气,按 **3 天 = 1 年** 换算。

具体: `起运岁数 = (距节气的小时数 / 24 / 3)` 取近似值。

### Python 计算示例

```python
from datetime import datetime, timedelta

# 出生时间(北京时间) —— 替换为命主实际出生时间
birth = datetime(YYYY, MM, DD, HH, mm)

# 阳年/阴年
year_stem_idx = (birth.year - 4) % 10
is_yang = year_stem_idx % 2 == 0  # 甲丙戊庚壬

# 男命阳年顺排,阴年逆排
gender = "male"  # or "female"
forward = (gender == "male") == is_yang

# 找出生日前后最近的两个节气(查万年历)
last_jieqi = datetime(YYYY, MM, DD, HH, mm)  # 上一个节气
next_jieqi = datetime(YYYY, MM, DD, HH, mm)  # 下一个节气

if forward:
    delta = next_jieqi - birth
else:
    delta = birth - last_jieqi

years_to_dayun = delta.total_seconds() / 3600 / 24 / 3
print(f"起运岁数 ≈ {years_to_dayun:.2f}")
```

## 真太阳时校正

中国大陆的钟表用北京时间(基于 120°E)。BaZi 严格说应该用**真太阳时**——基于出生地的实际经度。

### 校正公式

```
真太阳时 = 北京时间 + (出生地经度 - 120) × 4 分钟 + 均时差(EoT)
```

- 出生地偏西(经度 < 120) → 时间往前(更早)
- 出生地偏东(经度 > 120) → 时间往后(更晚)

**均时差(Equation of Time)**:太阳真实位置 vs 平均位置的差,一年内变化 ±16 分钟。一般近似:
- 4 月 / 6 月: ~+2 分钟
- 7-8 月: ~-1 到 -6 分钟
- 11 月: ~+16 分钟
- 2 月: ~-14 分钟

### 何时做校正

- 出生时辰落在分界点附近(尤其离整点 ±30 分钟)→ **必须校正**
- 出生地远离 120°E(西部省份、海南、新疆)→ **必须校正**
- 出生时辰明显在中间(比如下午 2 点 = 未时正中)→ 可以省略

### 注意

**真太阳时 vs 北京时间会改变时辰**——这是命理上的重大歧义。如果命主出生时刻在分界处,**两个方案都要算**,然后用回测确定哪个更准。

## 五行类象细节(传统命理常用)

| 五行 | 抽象类象 | 具体代表 |
|---|---|---|
| 木 | 生发、向上、生命力 | 树木、植物、纤维、纸、绿色、东方、春 |
| 火 | 升腾、表达、温暖 | 火、电、光、心血管、红色、南方、夏 |
| 土 | 承载、稳定、收纳 | 土地、不动产、消化系统、黄色、中央、四季末 |
| 金 | 收敛、肃杀、规则 | 金属、刀剑、肺、白色、西方、秋 |
| 水 | 流动、智慧、滋润 | 水、液体、肾、黑色、北方、冬 |

**重要**:**类象是抽象的**,不能直接等同于"穿什么颜色"或"住哪个方向"——这些是 D 级越界。

类象主要用于:
- 理解命主对外部世界的"取象偏好"
- 推论命盘缺失的五行(用神方向)
- 古籍中"妻为某行,父为某行"的原始判断

## 节气换算简表

BaZi 的"月"是**节气月**,不是日历月。月柱以节气划分:

| 月柱地支 | 起始节气 | 结束节气 | 阳历近似 |
|---|---|---|---|
| 寅(正月) | 立春 | 惊蛰前 | 2/4 - 3/5 |
| 卯(二月) | 惊蛰 | 清明前 | 3/6 - 4/4 |
| 辰(三月) | 清明 | 立夏前 | 4/5 - 5/5 |
| 巳(四月) | 立夏 | 芒种前 | 5/6 - 6/5 |
| 午(五月) | 芒种 | 小暑前 | 6/6 - 7/6 |
| 未(六月) | 小暑 | 立秋前 | 7/7 - 8/7 |
| 申(七月) | 立秋 | 白露前 | 8/8 - 9/7 |
| 酉(八月) | 白露 | 寒露前 | 9/8 - 10/7 |
| 戌(九月) | 寒露 | 立冬前 | 10/8 - 11/6 |
| 亥(十月) | 立冬 | 大雪前 | 11/7 - 12/6 |
| 子(十一月) | 大雪 | 小寒前 | 12/7 - 1/5 |
| 丑(十二月) | 小寒 | 立春前 | 1/6 - 2/3 |

**注意**:节气的具体时刻每年不同(误差 ±1 天),做命盘必须查具体年份的节气表。
