# 事业/工作 (career)

## 重点维度
- 格局 + 用神（决定"做什么类型的事最顺")
- 官杀的强弱与制化（官途/管理/受控环境的适配度）
- 食伤的强弱与位置（创造/输出/独立专业的适配度）
- 财星根气（变现能力）+ 印星（学习/资源/靠山）
- 当前大运对格局的修复或破坏

## 判断框架
- 身弱+官杀重 → 体制内压力大；身强+官杀清 → 适合管理
- 食伤生财 → 靠手艺吃饭；伤官见官 → 体制不顺，宜独立
- 印重身旺 → 学者/顾问；财重身弱 → 富屋贫人，需扶身

## 输出要求
- 给"做什么类型的事"，不要"要努力"这种废话
- 区分"结构性方向"（命盘底色）和"阶段性建议"（当前大运叠加）

## 好/坏开头示范

好的开头：
- "伤官见官，体制里的日子不会好过——但这话要先看格局再说，不是见了就断。"
- "此造食伤有力，财接得住，这是靠手艺吃饭的结构，问题只在'手艺'是什么。"
- "官杀清透，身又不弱，这类命格适合管人——但得先过了那道'能不能立住'的关。"

坏的开头：
- "你的事业运如下……"
- "根据你的格局，你适合从事……行业。"
- "你的官星代表你的事业和上司关系。"
