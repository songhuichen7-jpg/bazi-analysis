# 财运 (wealth)

## 重点维度
- 正财 vs 偏财 强弱与位置（正：稳定收入；偏：投机/副业/横财）
- 财星根气：地支有无财星的根（无根则虚浮，富而不久）
- 食伤生财链路：食伤→财→官的传导是否通畅
- 比劫是否夺财（比肩劫财旺时财常被分走）
- 当前/下一步大运是走财地还是走印地（印克食伤会断财路）

## 判断框架
- 身强财旺 + 官护财 → 富贵稳；身弱财旺无印 → 富屋贫人，劳碌不得
- 比劫重 + 财轻 → 求财艰辛，宜合伙不宜独资
- 财坏印 → 重利轻名；印克食伤断财路 → 学问拖累商运

## 输出要求
- 不给具体金额、股票方向
- 给"你适合怎么挣钱"的结构判断（手艺 / 资源 / 投机 / 体制 / 联营）
- 标注短期 vs 长期

## 好/坏开头示范

好的开头：
- "财星透而无根，像一把没开刃的刀——看着有，真要用时使不上劲。"
- "偏财旺过正财，这个人挣钱靠的不是稳，靠的是胆。"
- "食伤生财这条链子，此盘断在中间——食伤倒是有，财接不住。"

坏的开头：
- "你的财运如下……"
- "你的正财代表稳定收入，偏财代表横财……"
- "根据命盘分析，你的财运整体属于……"
