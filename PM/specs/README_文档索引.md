## 产品概况

一句话：用 AI 做一个"八字版 MBTI"——用户输入生日，3 秒拿到专属人格卡片，可以分享、合盘、深度解读。

- **裂变层**（获客）：人格卡片 + 合盘卡片 → 社交分享
- **核心产品**（留存/付费）：AI 专栏报告 + AI 自由对话
