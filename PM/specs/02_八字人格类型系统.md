## 一、设计目标

1. 用户输入生日 → **确定性**映射到 20 种类型之一
2. 每个类型有**典籍断词依据**
3. 标签和描述有**梗感 + 分享欲**
4. 类型分布**相对均匀**（每种约 5%）
5. 标签风格：**精准重构 × 场景化自嘲 × 不装不端**

---

## 二、为什么是 20 型

### 方案对比

| 数量 | 方法 | 优势 | 劣势 |
|------|------|------|------|
| 5（五行） | — | 极简 | 太粗，每种 20%，差异化不足 |
| 10（十天干） | v2 方案 | 映射清晰 | 10% 撞型率高，朋友圈容易重复 |
| **20（天干 × 身强弱）** | **v3** | **典籍直接支持，接近 MBTI 16型，传播差异化好** | 需要身强弱判定算法 |
| 40+（天干 × 四季） | — | 极高差异化 | 太复杂，文案量爆炸 |

### 20 型的核心理由

1. **命理正当性最强** -- 渊海子平原文："日干弱则退缩怕羞，日干强则妄诞执一"，**同一天干强弱完全两种人**
2. **滴天髓支持** -- 每个天干论述中都按旺/衰条件做了不同描写（如甲木"旺极宜泄" vs "失时就衰，根气收敛"）
3. **三命通会框架** -- "五行 × 旺相/太过/不及" ≈ 15 种变体，20 型在此基础上更精确
4. **传播学验证** -- 20 型→210 合盘组合，接近 MBTI 的 136 组，差异化充分
5. **映射仍然确定性** -- 日柱天干 + 身强弱评分 = 固定结果
6. **与 v2 兼容** -- 10 个天干标签保留为"底层"，身强弱作为"变体修饰"

## 三、映射逻辑：日主天干 × 身强/身弱

### 映射流程

```
用户输入生日 → 排盘 → 四柱八字
                        ↓
          ┌─ 日柱天干（甲/乙/丙/丁/戊/己/庚/辛/壬/癸）
          │
          └─ 身强/身弱评分 → 强 or 弱
                        ↓
              10 天干 × 2 强弱 = 20 种类型
```

### 身强/身弱判定算法

基于子平真诠"得时不旺失时不弱"原则，综合四维评分：

| 维度 | 权重 | 计分规则 |
|------|------|---------|
| **得令**（月支） | 3 分 | 日主五行在月支当旺/当令 = +3；月支生日主 = +1；月支克/泄日主 = 0 |
| **得地**（地支根气） | 每根 2 分 | 日支/年支/时支中，地支藏干含日主同五行 = +2/根 |
| **得生**（印星） | 每个 1 分 | 天干中出现生日主的五行（印）= +1/个 |
| **得助**（比劫） | 每个 1 分 | 天干中出现与日主同五行（比劫）= +1/个 |

**阈值**：总分 ≥ 4 分 = 身强；＜ 4 分 = 身弱

**合/冲修正**：
- 日主根被冲：该根不计分
- 天干合化（非本五行）：该干不计相应分

此算法可编程实现，同一生日永远得到相同结果。

### 分布预估

经验上，约 45-55% 的人身强，45-55% 身弱。10 天干各约 10%，叠加后每种约 5%，分布合理。

## 四、20 型命名体系

### 四层命名结构

```
Layer 0：传播名（社交货币层）   → "春笋"          ← 用户看到的第一个词，类似MBTI"快乐小狗"
Layer 1：底色名（命理意象层）   → "参天木命"
Layer 2：人格标签（梗感层）     → "参天型"
Layer 3：一句话（场景化重构）   → "越压越往上长"
强弱修饰：显示在标签旁          → "绽放" or "蓄力" 以视觉标记区分
```

### 传播名设计原则

传播名是用户在社交平台分享时最先看到的标签，需满足：
1. **一秒可懂**：具象名词，无需解释
2. **高分享欲**：适合做头像、表情包、社交货币
3. **情绪光谱均衡**：暖系/酷系/怪系/利落系混搭
4. **命理映射可辩护**：意象与天干本性、绽放/蓄力状态关联
5. **不重叠**：同五行内绽放/蓄力有明确差异，跨五行不撞语义槽位

### 产品化命名：绽放型 vs 蓄力型

**面对用户时不使用"身强/身弱"**。产品化命名：
- 身强 → **绽放型**（能量外放、正在输出期）
- 身弱 → **蓄力型**（能量内收、正在积累期）

在卡片和分享页面上显示为："甲·绽放" / "甲·蓄力"，而非"身强/身弱"。

## 五、每个类型的完整文案结构

每个类型需要以下元素（由主 agent 生成 → 评审定稿）：

```
1. 底色名（≤4字）
2. 人格标签（≤5字）
3. 一句话（≤20字）
4. 三个子标签（各≤6字）
5. 金句（2句话，可独立截图分享）
6. 正面重构句（"这不是XX，是XX"格式）
7. 底色描述（1句，引用古籍意象）
```

### 强弱差异化原则

| 维度 | 绽放型 | 蓄力型 |
|------|--------|--------|
| 标签气质 | 外放、主动、扛压、我行我素 | 内蓄、借力、积累、待时而动 |
| 一句话 | 偏向"我就是这样" | 偏向"我知道我需要什么" |
| 金句 | 强调"输出"和"不怕" | 强调"蓄力"和"等到时候" |
| 重构句 | "这不是XX，是XX（正面力量）" | "这不是XX，是XX（智慧等待）" |

**关键**：蓄力型不是"弱者"，是"还在积累/善于借力"。滴天髓原文中弱甲"得火而发荣"、弱辛"热则喜母寒则喜丁"，都是**知道自己需要什么**的智慧。

**产品声明**：绽放/蓄力为体验型分型，旨在帮助用户理解自身能量节奏，非传统命理学的身旺身弱判定。典籍引用为意象参考，不做命理论断。

## 八、标签风格规范

**是什么**：像一个很懂你的朋友，用你可能说不出口但心里认同的话，帮你把"以为是缺点的东西"翻面。

**不是什么**：

| 不是 | 反例 |
|------|------|
| 心理测评 | "你具有较强的内省倾向" |
| 算命先生 | "你命犯七杀，主有灾厄" |
| 励志鸡汤 | "相信自己，你一定可以" |
| SBTI式骂人 | "你就是个废物" |
| 蓄力=弱者 | "你比较弱，需要帮助" ← **绝对禁止** |

**蓄力型标签红线**：蓄力永远不是"弱"或"差"，而是"正在积累"、"善于借力"、"知道自己需要什么"。滴天髓对每个蓄力状态的描写都是正面的——甲"得火而发荣"，丁"如有嫡母可秋可冬"。

**文案公式**：

```
绽放型：[场景化描述] + [正面力量重构]
"越压越往上长" → 意象（春笋）+ 重构（压力=生长力）

蓄力型：[传播名意象] + [蓄势重构]
"别催，在攒大招" → 意象（橡子）+ 重构（等待=蓄势）
```
