## 一、标签结构


| 位置  | 类型     | 来源                         | 长度  | 示例     |
| ------- | ---------- | ------------------------------ | ------- | ---------- |
| 标签1 | 性格标签 | v4.0定稿中选最佳1个（不动）  | 2-5字 | 边冲边享 |

### 命势标签设计原则（v6.0·第一性原理版）

#### 场景还原

用户输入生日 → 3秒出卡片 → 看到三个标签 → 截图发朋友圈。

```
🌱 春笋 · 天生享乐家
性格：冲上去再说
关系：人缘自己来
事业：认识人就有钱
```

**用户3秒内的唯一反应：「说的是不是我？」**

#### 三列各自回答什么问题


| 列       | 用户心里的问题            | 好标签 = 具体画面             | 坏标签 = 抽象废话  |
| ---------- | --------------------------- | ------------------------------- | -------------------- |
| **性格** | 我平时是怎样的人？        | 一个能对号入座的行为场景      | 笼统形容词         |
| **关系** | 我在感情/社交里是怎样的？ | 一个能对号入座的感情画面      | "X缘""X甜"等空泛词 |
| **事业** | 我怎么赚钱/在职场怎样？   | 一个能对号入座的工作/赚钱画面 | "X运""X势"等抽象词 |

#### 3秒测试（每条标签必须同时过三关）

1. **辨识度**：用户能秒懂这是在说性格/关系/事业哪个
2. **画面感**：脑子里能浮现一个具体场景
3. **对号入座**：用户能点头说"对这就是我"

#### 十神 × 三列 = 用户想看到的具体答案


| 十神 | 性格：我是怎样的人？ | 关系：我的感情/社交？ | 事业：我怎么赚钱？ |
| ------ | ---------------------- | ----------------------- | -------------------- |
| 食神 | 我享受生活的方式     | 我讨人喜欢的方式      | 我靠什么享受还赚钱 |
| 伤官 | 我表达自己的方式     | 我吸引人的独特方式    | 我的才华怎么变成钱 |
| 正官 | 我守规矩的方式       | 我找对象的风格        | 我在正规路上怎么升 |
| 七杀 | 我面对压力的方式     | 我在感情困难中怎样    | 我怎么在逆境中翻盘 |
| 正印 | 我学习/积累的方式    | 谁在帮我/疼我         | 学习怎么帮我赚更多 |
| 偏印 | 我跟别人不一样的地方 | 我怎么遇到特别的人    | 我在冷门路上怎么赚 |
| 正财 | 我踏实做事的方式     | 我怎么经营长期关系    | 我怎么稳定攒钱     |
| 偏财 | 我社交/处世的方式    | 我的异性缘/社交魅力   | 人脉怎么帮我赚钱   |
| 比肩 | 我独立/不从众的方式  | 我一个人的时候怎样    | 我适合独立做什么   |
| 劫财 | 我争/抢/竞争的方式   | 我在感情竞争中怎样    | 我在职场竞争中怎样 |

#### 硬规则

1. **标签在说"你"** — 不用"X的人Y"泛指句式（像人生格言），而是隐含"你"为主语
   - ✗ "温柔的人桃花多" → 像鸡汤金句
   - ✓ "越温柔桃花越多" → 在说"你"
2. **正面化** — 没有坏运势，有挑战也用积极框架
3. **不用命理术语** — 八字/十神/天干等不出现在标签中
4. **不使用动物意象** — 命势列只写运势本身，不塞"壳/刺/触手/鼓"等传播名梗
5. **有典籍映射** — 天干×十神的经典论述可追溯（详见 specs/10_典籍命格参考.md）
6. **允许合理重复** — 区分度来自命理差异（五行×十神），不追求200条完全不同
7. **stigma禁用词清单继续生效** — R1-R8积累的禁用词（见下方）不重新踩坑

### 区分度策略（古籍支撑）

基于 `specs/10_典籍命格参考.md` Part C 交叉矩阵分析：


| 区分层级     | 数量         | 区分方式                       | 示例                               |
| -------------- | -------------- | -------------------------------- | ------------------------------------ |
| **十神主题** | 10种         | 每个十神有固定运势主题         | 食神→口福/享福，伤官→桃花/才华   |
| **五行底色** | ×5 = 50种   | 木/火/土/金/水各有运势风格     | 木食神→向上享福，金食神→精工享福 |
| **阴阳微调** | ×2 ≈ 80种  | 阳干大开/阴干柔细              | 甲食神→大口享福，乙食神→小确幸福 |
| **绽蓄方向** | ×2 ≈ 100种 | 绽放=当下/主动，蓄力=后期/积蓄 | 绽放→桃花旺，蓄力→迟来桃花       |

最终：约100种有意义区分 + 约100种允许微调/合理近似。不追求200条完全不同。

### R1-R8 stigma禁用词清单（继续生效）

- 命理术语：壬水既济、金水伤官、藤萝系甲、羊刃驾杀等
- 封建迷信：四方来财、步履生金、XX型财运（批量）
- 宿命论：命中注定、不争却被选中、飘到谁算谁
- 性别刻板：撒娇得宠、示弱得宠
- 过度承诺：玩着赚钱、暖到哪赚到哪
- 暗黑感：毒液、蓄势必杀、一出手见血
- 苦/患难：先苦后甜、苦尽甘来、患难见真情
- 贵人/靠山过度：贵人自来、自带靠山感（控制密度）
- "旺"过度：XX旺（可用但不超3-4处）
