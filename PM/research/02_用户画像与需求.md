# 用户画像与需求分析

> 最后更新：2026-04-17
> 数据来源：2025年轻人玄学消费报告（36氪/行业报告智库）、腾讯新闻、创业邦、AI 百宝等

---

## 一、宏观用户画像

### 1.1 核心人群特征

| 维度 | 数据 |
|------|------|
| **核心年龄段** | 18-30 岁（占在线命理消费 70%+） |
| **性别分布** | 女性略多（约 52%），但男性需求不可忽视 |
| **地域** | 一二线城市为主，下沉市场正在快速增长 |
| **教育水平** | 高学历群体为主力 -- 玄学消费与"反智"无关 |
| **消费力** | 60% 年玄学消费 < 500 元；20% > 1000 元 |
| **人均单次付费意愿** | 约 100 元以内 |

### 1.2 性别差异

| 维度 | 女性 | 男性 |
|------|------|------|
| 玄学热情评分 | **6.73** / 10 | **5.95** / 10 |
| 核心关注 | 事业爱情"大女主"双丰收 | 健康、家庭、人际的"全家平安" |
| 行为特征 | 更愿意分享、讨论、社交互动 | 更偏向独自查看，不太主动分享 |
| 付费倾向 | 手串、塔罗体验、详细报告 | 风水摆件、专项咨询 |

### 1.3 年龄梯度

| 年龄段 | 典型行为 | 消费形态 | 产品启示 |
|--------|---------|---------|---------|
| **18-22（00 后）** | 转发锦鲤、做测试、社交分享 | 几乎零成本（免费为主） | 免费 + 极强社交货币属性 |
| **23-27（95 后）** | 购买转运珠、深度塔罗体验 | 小额付费（50-200 元） | 可承受单次付费但要"值" |
| **28-35（90 后）** | 研究风水布局、请专业咨询 | 中等付费（200-1000 元） | 深度内容和对话功能的付费用户 |
| **35+** | 风水、择日、合婚等传统需求 | 高额付费但线上占比低 | 非核心目标群体 |

## 二、用户分层模型

基于使用深度和付费意愿，将用户分为四个层级：

### Tier 1：好奇尝鲜型（约 50%）

| 维度 | 描述 |
|------|------|
| **画像** | 看到朋友分享 → 点进来测一下 → 5 分钟内离开 |
| **核心需求** | 快速获得一个**有趣的、可分享的**结果 |
| **付费意愿** | 几乎为零 |
| **产品价值** | 传播的源头和放大器 -- 他们不付费但帮你传播 |
| **设计要点** | 输入生日即出结果，无需注册；结果页必须有"截图时刻" |

### Tier 2：自我探索型（约 30%）

| 维度 | 描述 |
|------|------|
| **画像** | 对自己的性格、运势有真实好奇，愿意花 10-15 分钟看完报告 |
| **核心需求** | **"被精准描述"的体验** -- 感觉"这说的就是我" |
| **付费意愿** | 9.9-49 元/次 |
| **产品价值** | 核心用户群 -- 他们构成口碑和收入的主体 |
| **设计要点** | 报告需要具体、有据、避免巴纳姆效应；叙事体 > 列表式 |

### Tier 3：深度咨询型（约 15%）

| 维度 | 描述 |
|------|------|
| **画像** | 遇到人生困惑（感情/事业/择业），想获得具体建议 |
| **核心需求** | **对话式"命理顾问"体验** -- 能针对具体问题给方向 |
| **付费意愿** | 49-199 元/次 |
| **产品价值** | 高 ARPU 用户 -- token 消耗大但付费能力强 |
| **设计要点** | 自由对话功能；能记住上下文；像"懂命理的朋友"而非"客服机器人" |

### Tier 4：命理爱好者（约 5%）

| 维度 | 描述 |
|------|------|
| **画像** | 懂一些命理知识，想验证或深入 |
| **核心需求** | **专业的排盘和解析**，能看到原始命盘结构 |
| **付费意愿** | 为工具型功能付费（月费制） |
| **产品价值** | 社区种子用户 -- 他们能成为口碑传播的"内行"（引爆点理论） |
| **设计要点** | 展示完整排盘、典籍引用原文、支持多流派对比 |

### 用户转化漏斗

```
Tier 1（好奇尝鲜）→ 分享传播 → 拉入新的 Tier 1
      ↓
Tier 2（自我探索）→ 付费看完整报告
      ↓
Tier 3（深度咨询）→ 付费对话
      ↓
Tier 4（命理爱好者）→ 成为社区贡献者 / KOC
```

## 三、用户核心诉求排序

### 3.1 关注领域排名

| 排名 | 领域 | 占比 | 备注 |
|------|------|------|------|
| 1 | **事业运** | 76.5% | "先搞钱再谈爱" |
| 2 | **财运** | 74.9% | 与事业高度相关 |
| 3 | **爱情运** | 49.6% | 传统认知中的第一需求实际排第三 |
| 4 | **健康** | 约 40%（男性更高） | 男性差异化需求 |
| 5 | **人际 / 家庭** | 约 35% | 男性差异化需求 |

**产品启示**：内容优先级应该是 **事业 > 财运 > 感情**，而非传统算命产品的"感情优先"。

### 3.2 用户期待的体验特征

| 期待 | 具体含义 | 反面教材 |
|------|---------|---------|
| **具体而不笼统** | "你在 25-27 岁会遇到事业转折期" > "你未来事业运不错" | 巴纳姆效应 = 信任杀手 |
| **有趣可分享** | 结果页有视觉冲击力、一句话标签、社交属性 | 干巴巴的文字报告无人分享 |
| **不被说教** | 像朋友聊天 > "大师开示" | "PUA 感""像在训人" |
| **隐私安全** | 出生日期是敏感数据，需要信任感 | 要求手机号 + 真名 = 劝退 |
| **快速出结果** | 输入后 3 秒内有反馈 | 加载等待 > 30 秒 = 流失 |
| **有文化底蕴** | 引经据典 > AI 编造 | "典籍说…"但查不到出处 |

## 四、用户使用玄学的心理分析

### 4.1 心理机制

| 心理需求 | 命理产品如何满足 | 学术依据 |
|---------|---------------|---------|
| **控制感缺失** | 命理框架给不确定的未来提供"可控幻觉" | 控制感理论（Locus of Control） |
| **身份确认** | "我是 XX 命"提供身份标签和归属感 | 社会认同理论 |
| **情绪宣泄** | "命里有此一劫"将困境外归因，缓解自责 | 归因理论 |
| **决策辅助** | 不做决定但给方向感，降低决策焦虑 | 决策疲劳理论 |
| **社交连接** | "你什么命？"开启对话，创造共同话题 | 社交货币理论 |

### 4.2 使用场景

| 场景 | 频率 | 情绪状态 | 需求深度 |
|------|------|---------|---------|
| 朋友分享看到，好奇点进 | 一次性 | 轻松好奇 | 浅（Tier 1） |
| 生日前后想了解新一年运势 | 年度 | 期待 + 微焦虑 | 中（Tier 2） |
| 感情出问题想知道原因 | 事件驱动 | 焦虑 / 伤心 | 深（Tier 3） |
| 求职 / 跳槽前想看方向 | 事件驱动 | 迷茫 / 焦虑 | 深（Tier 3） |
| 年初规划 / 节气转折 | 周期性 | 仪式感 | 中（Tier 2） |
| 闲聊中"顺便测一下" | 随机 | 轻松 | 浅（Tier 1） |

## 五、对产品设计的直接指导

### 5.1 必须做到

1. **免费层足够有分享价值** -- Tier 1 是传播引擎，他们的体验决定产品能否裂变
2. **事业和财运内容优先** -- 而非传统的"桃花运优先"
3. **输入极简化** -- 只要求出生年月日时 + 性别（时辰可选），不强制注册
4. **结果页有"一句话标签"** -- 类似 SBTI 的标签设计，给用户"社交货币"
5. **叙事 > 术语** -- 用故事讲命盘，不是用表格列数据

### 5.2 必须避免

1. **不要一上来就要手机号** -- 信任门槛太高
2. **不要做"付费才能看完"的钓鱼模式** -- 用户对这种模式已经厌倦
3. **不要用说教口吻** -- "你应该 XX""你必须 XX"是禁忌
4. **不要让 AI 假装"大师"** -- 年轻人对权威人设本能反感
5. **不要只做一次性报告** -- 必须有流年 / 大运等持续内容

### 5.3 差异化机会

| 竞品痛点 | 我们的对策 |
|---------|----------|
| AI 编造 | 典籍直引 + 标注出处 |
| 模板感强 | 叙事体 + 对话式 |
| 排盘不准 | 双引擎校验（前端 + MCP） |
| 只有报告没有对话 | 支持自由对话（Tier 3 付费） |
| 只关注感情 | 事业 / 财运优先 |
| 无持续价值 | 流年运势 + 节气内容 + 大运切换提醒 |
