---

## 背景

朋友完成概念设计，现在需要把概念变成可执行的产品规格。

**v2 修正**：v1 只关注了裂变层（卡片/合盘），遗漏了核心产品（AI 解读 + 对话）。v2 全面覆盖两层架构。

## v1 → v2 修正记录

| 修正项 | v1 | v2 |
|--------|----|----|
| 类型数量 | 8 型（2 种合并天干） | 10 型（十天干 1:1） |
| 产品范围 | 只有裂变层 | 裂变层 + 核心产品层 |
| AI 功能 | v2 再做 | MVP 就包含 |
| 标签风格 | 安全、正确 | 梗感、场景化自嘲 |
| 文案生成 | 手写 | AI 生成 + 人工筛选 |
| 专栏报告 | 无 | 5 份（命格/事业/流年/感情/相处指南） |
| 自由对话 | 无 | 多轮 LLM 对话 + RAG |
| 合盘矩阵 | 8×8（36 种） | 10×10（55 种） |
