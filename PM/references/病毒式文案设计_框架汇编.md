# 病毒式文案设计：框架汇编

> 汇编来源：Schwartz《Breakthrough Advertising》、2026 Viral Hook 心理学研究、SBTI 文案拆解
> 对产品的价值：指导标签文案和分享文案的病毒性设计

---

## 一、Schwartz 认知阶梯（五层认知模型）

Eugene Schwartz 的核心洞察：**你不能创造欲望，只能引导已有的欲望。**

### 五层认知与对应文案策略

| 层级 | 用户状态 | 文案策略 | 在我们产品中 |
|------|---------|---------|------------|
| **完全无感知** | 不知道自己有需求 | 用情感切入，不卖产品 | 朋友圈看到别人的卡片 -- 被视觉吸引但还没有"我也想测"的念头 |
| **问题感知** | 知道自己迷茫/好奇但不知道有解决方案 | 共鸣痛点，引入品类 | 分享文案："你知道自己是什么底色吗？"-- 唤醒自我认知需求 |
| **方案感知** | 知道有测试/算命类产品但不知道我们 | 差异化，说清楚"我们不一样" | 标签下的典籍引用 -- "和普通 AI 算命不一样，这个有古籍依据" |
| **产品感知** | 知道我们但没行动 | 消除疑虑，给推力 | "只需要输入生日，3 秒出结果" |
| **完全感知** | 已经准备行动 | 给 offer、给紧迫感 | 输入框 + 大按钮 -- 零阻力 |

### 对我们的启示

大多数用户是从**完全无感知**进入的（在朋友圈偶遇）。这意味着：
1. **分享卡片就是"Level 1 文案"** -- 不能直接卖产品，要先用视觉和情感抓住注意力
2. **从感知到行动最多允许 2 步** -- 看到卡片 → 点击链接 → 输入生日。多一步就流失

## 二、2026 病毒 Hook 七大心理触发器

基于 2025-2026 年对 1000+ 爆款内容的数据分析：

### 触发器 1：模式打断（Pattern Interrupt）

**原理**：打破用户预期的信息格式，迫使大脑重新聚焦。

| 在我们产品中 | 示例 |
|------------|------|
| 标签命名 | "先压后起型" -- 预期是"XX 命"，实际是动词短语 |
| 合盘标签 | "破局搭子" -- 命理合盘的结果不是"天作之合"，而是"搭子" |

### 触发器 2：好奇缺口（Curiosity Gap）

**原理**：给出结果但不给原因，制造"我想知道为什么"。

| 在我们产品中 | 示例 |
|------------|------|
| 分享文案 | "我是山路火种？？" -- 看的人想知道"这是啥意思" |
| 合盘 CTA | "看看我们是不是这种搭子" -- 想知道自己和朋友是什么关系 |

### 触发器 3：反直觉（Contrarian）

**原理**：挑战常识，触发认知失调，迫使人重新评估。

| 在我们产品中 | 示例 |
|------------|------|
| 正面重构 | "这不是坏命，是先压后起" -- 挑战"命不好 = 不幸"的常识 |
| 颠覆认知 | "不是选择困难，是你看得太透了懒得选" -- 把缺点重构为优势 |

### 触发器 4：超具体（Specificity Effect）

**原理**：越具体越可信。"97.3%" 比 "大约 100%" 有说服力。

| 在我们产品中 | 示例 |
|------------|------|
| 差异化描述 | "这种命格在同龄人中约占 12%" -- 而非"比较少见" |
| 时间节点 | "你的 27-29 岁是关键转折期" -- 而非"未来运势不错" |

### 触发器 5：时间压缩（Timeframe Tension）

**原理**：短时间 + 大结果 = 多巴胺飙升。

| 在我们产品中 | 示例 |
|------------|------|
| 交互提示 | "输入生日，3 秒看到你的命盘底色" |
| 分享文案 | "3 秒就被说中了" |

### 触发器 6：情绪唤起（Emotional Arousal）

**原理**：高唤醒情绪（惊讶、敬畏、幽默）促进分享。

| 高唤醒（好） | 低唤醒（差） |
|------------|------------|
| "卧槽这也太准了" | "挺准的吧" |
| "这什么鬼标签哈哈哈" | "嗯，有道理" |
| "等一下，这怎么知道的" | "大概就是这样" |

### 触发器 7：社交证明 + FOMO

**原理**：看到别人都在做 → 我也想做。

| 在我们产品中 | 示例 |
|------------|------|
| 朋友圈效应 | 当多个朋友的卡片出现在同一天的朋友圈 → FOMO |
| 统计展示 | "已有 XX 人查看了自己的命盘底色"（后期功能） |

## 三、标签文案的黄金公式

综合以上框架，标签文案应该满足：

### 公式：意象标签 + 正面重构金句

```
[意象标签] -- 用具体物象（山路火种、灵感火花）替代抽象概念
     +
[一句话描述] -- 用反直觉的正面重构让用户感到"被看见"
     +
[金句] -- 可独立截图分享的、有情绪唤起的表达
```

### 检验标准

一个好的标签文案在发到朋友圈时应该触发以下反应链：

```
看到朋友的分享卡片
    ↓
"这什么东西？"（Pattern Interrupt）
    ↓
"山路火种是什么意思？"（Curiosity Gap）
    ↓
"等等，先压后起，这不就是我吗？"（Emotional + Credible）
    ↓
"我也要测一下"（Action）
    ↓
"我的结果好有意思 → 发朋友圈"（Social Currency + Public）
```

## 四、分享文案的写法规范

### 分享文案的三种场景

| 场景 | 需要的文案 | 风格 |
|------|----------|------|
| **微信朋友圈** | 配图文案（用户发卡片截图时的配文） | 短、自嘲、引好奇 |
| **微信分享卡片** | JS-SDK 配置的标题 + 描述 | 标题抓眼，描述给信息 |
| **合盘邀请** | 邀请好友来合盘的 CTA | 有趣、有行动指令 |

### 示例（仅为风格参考，非最终文案）

**朋友圈配文风格**：
- "我居然是山路火种?? 说我压力越大越能顶住…好像是这样没错"
- "查八字人格图鉴 -- 这比 MBTI 准多了吧"
- "8 种底色，我是第 X 种，你呢？"

**合盘邀请风格**：
- "你扛住，我想招 -- @XX 看看我们是不是这种搭子"
- "我和你五行居然是 XX 局？来查查看"

### 文案红线

| 绝对不能出现 | 原因 |
|------------|------|
| "预测你的命运" | 合规红线 |
| "你命不好" / 任何负面标签 | 没人分享负面结果 |
| "大师""算命" | 社交风险太高 |
| 强制分享才能看结果 | 微信会封 |

## 五、"全是好听的"策略的理论支撑

你朋友说"合盘全是好听的语言就行了"。这个直觉是对的，理论依据：

1. **Schwartz**：用户的欲望是"被肯定""被理解"，不是"被批评"。引导已有欲望。
2. **STEPPS - Social Currency**：人们分享让自己看起来好的内容。没人分享"你俩不合"。
3. **SUCCESs - Emotional**：正面情绪 > 负面情绪的分享率。
4. **Co-Star 的教训**：Co-Star 偶尔发负面推送引发争议，但争议不等于留存。

**具体做法**：
- 合盘结果全部是**正面框架**（不同程度的好），没有"不合"
- "不那么合"的情况用**成长性语言**替代：不是"你们不合"，而是"你们是互相磨砺型"
- 所有描述强调**互补**和**共同成长**，而非匹配程度打分
